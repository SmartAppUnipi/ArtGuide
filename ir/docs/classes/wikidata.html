Private getEntityRootPath

  • getEntityRootPath(wikidataId: string): Promise<Array<string>>
    • Defined in ir/src/wiki/wikidata.ts:222

    Given a WikiData entity returns the list of the labels for properties 'InstanceOf' (P31) and 'SubClassOf' (P279) until the WikiData root.

    This function exploits SparQL to perform a single HTTP request.

    The returned list is unordered.

    Parameters

    • wikidataId: string

      The WikiData entity id.

    Returns Promise<Array<string>>

    An array with the labels corresponding to the properties values (eg. for Pisa Tower => ["bell tower", "tower", "architectural structure", "work", "construction", ... ])
